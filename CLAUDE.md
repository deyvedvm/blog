## Architecture

**Astro 5** static blog with **Tailwind CSS 4**, deployed on Netlify. Content is Markdown with frontmatter, managed via Astro Content Collections (new Content Layer API with glob loaders).

### Site configuration

`src/config.ts` — all site constants: author info, social links, menu items, Disqus shortname, Google Analytics ID, posts per page.

### Content

`src/content/config.ts` — Zod schemas for both collections using `glob` loaders (important: uses new Content Layer API, not legacy collections, to avoid the `slug` reserved-field conflict).

- `src/content/posts/*.md` — Blog posts. Required frontmatter: `title`, `date`, `slug`, `description`. Optional: `draft`, `category`, `tags`, `socialImage`.
- `src/content/pages/*.md` — Static pages (about, contacts).
- `public/media/` — Post images. `public/photo.jpg` — author photo.

**Draft posts** are filtered out with `getCollection('posts', ({ data }) => !data.draft)` in index, tag, and category pages. Post pages are built for ALL posts regardless of draft status.

**Slugs**: `post.data.slug` (from frontmatter) is used for all URL generation. `post.id` is the filename-based ID and is NOT used for routing.

### Layouts & Components

- `src/layouts/Base.astro` — HTML shell, `<head>`, OG meta tags, Google Analytics gtag
- `src/layouts/Blog.astro` — Two-column layout: sticky sidebar (lg) + main slot
- `src/components/Sidebar.astro` — Author photo/bio, navigation (from `MENU`), social icons, copyright
- `src/components/PostCard.astro` — Post summary card for feeds
- `src/components/Pagination.astro` — Prev/next navigation
- `src/components/Comments.astro` — Disqus via vanilla JS `define:vars` (no React needed)
- `src/components/TagList.astro` — Tag pills linking to `/tag/{slug}`

### Routing

| URL | File |
|---|---|
| `/` | `src/pages/index.astro` (page 1) |
| `/page/[n]` | `src/pages/page/[page].astro` |
| `/[slug]` | `src/pages/[slug].astro` (posts) |
| `/tag/[tag]` | `src/pages/tag/[tag].astro` |
| `/category/[cat]` | `src/pages/category/[category].astro` |
| `/tags` | `src/pages/tags/index.astro` |
| `/categories` | `src/pages/categories/index.astro` |
| `/about`, `/contacts` | static pages fetching from `pages` collection |
| `/rss.xml` | `src/pages/rss.xml.ts` |

Sitemap is auto-generated by `@astrojs/sitemap`. Netlify publishes from `dist/`.

### Styling

Tailwind CSS 4 via `@tailwindcss/vite` Vite plugin (no `tailwind.config.js`). Config is in `src/styles/global.css`. Typography for post body uses `@tailwindcss/typography` (`prose` class). Dark mode uses `prefers-color-scheme` media query automatically.
